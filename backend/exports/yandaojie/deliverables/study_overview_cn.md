# 研道街 · 辩护式评估研究 — 研究说明（给合作教师 / 伦理审查）

> 一页说明。带【】的为待你按实际情况填写的占位。

## 一、研究背景与目的
我们在小学课堂部署了一个 AI 辅导系统，它让学生先写学习反思，再以追问的方式要求学生为自己的答案“答辩”（说清“为什么”）。
本研究要回答：**这种“辩护式评估”能否揭示选择题/对答案式评分结构性漏掉的理解缺口**——尤其是“**结论对、但机制错**”（例如：判断“化学变化”正确，理由却是“味道一样”）；以及它**在哪些学科有效、在哪些退化**。

## 二、数据来源
数据已于【时间/学校/年级】在真实课堂采集完毕，规模：**50 名学生、140 个学习会话、3 个学科（数学 / 科学 / 英语）、551 轮答辩对话**。分析使用的是这些**已采集、已【匿名化方式：学生以编号表示，不含姓名等可识别信息】**的对话记录。教师参与者**不与学生本人接触**，只阅读这些记录。

## 三、两个子研究与教师参与
**研究一 · 辩护深度标注（核心）**
- 每个学科招募 **3 位本学科在职教师**，对本学科的辩护记录做两项独立标注：
  ① **MC视角**：若只看学生说出的答案/结论（选择题式评分），是否会判为“已掌握”？
  ② **辩护深度**：看完整段答辩后，学生的真实理解属于 **扎实 / 对标签错机制 / 空洞或猜对 / 完全不懂 / 难判断**。
- 正式标注前先做 **8 题对齐练习（norming）**，统一标准。预计每位教师约 **【X】 小时**。

**研究二 · 机制缺口报告（设计实验，后续/可选）**
- 招募**更大规模的本学科教师样本**，对比两种呈现——“仅掌握分” vs “机制缺口报告（指出学生暴露出的具体错误机制）”——看哪种更能帮老师**识别真懂/假懂、并定下一步教学**。预计每位约 **【X】 分钟**。

## 四、教师在研究一需要做什么
1. 阅读《编码手册》与《评分 SOP》（随评分包提供）。
2. 独立完成 8 题对齐练习 → 与主持人核对、讨论分歧 → 统一理解。
3. 在《标注表》逐条独立标注（三位老师互不商量）。
全程仅阅读匿名对话记录，约 **【X】 小时**，可分次完成。
